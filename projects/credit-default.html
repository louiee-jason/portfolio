Credit Default Prediction

A machine learning pipeline built to predict customer credit default risk using financial data, incorporating robust resampling methods to handle extreme class imbalances.

Project Overview

In the financial industry, accurately predicting which customers are likely to default on their credit is critical for minimizing losses and managing risk. However, real-world financial datasets are notoriously noisy and highly imbalanced—the number of customers who default is typically much smaller than those who pay on time.

This project tackles this challenge head-on by building a robust machine learning model that predicts customer credit default risk, employing advanced data balancing techniques to ensure the model learns to identify high-risk individuals effectively.

Methodology

The project followed a rigorous end-to-end data science lifecycle:

  • Data Preprocessing: Extensive cleaning of noisy financial data and engineering new features to better capture financial behavior.
  • Class Imbalance Handling: Applied Resampling Methods (like SMOTE/Oversampling) to balance the dataset, preventing the model from simply predicting the majority class (non-default).
  • Model Training: Trained and evaluated multiple models, focusing heavily on Random Forest for its robustness against overfitting and ability to handle non-linear relationships.
  • Model Evaluation: Because of the imbalanced nature of the problem, standard accuracy was insufficient. The model was evaluated using strictly relevant metrics: ROC-AUC, Precision, Recall, and F1-score.

Results & Key Insights

  • Model Performance: The Random Forest model achieved significantly improved recall for the default class, meaning the bank can successfully detect a much higher percentage of true high-risk customers without excessive false alarms.
  • Behavioral Insight: Feature importance analysis revealed that customers exhibiting a combination of higher credit utilization and lower income have a drastically higher probability of defaulting.

Business Impact

  • Risk Mitigation: Directly helps financial institutions reduce monetary losses by accurately flagging high-risk applications before credit is issued.
  • Strategic Decision Making: Provides a data-driven foundation for credit approval boards, moving away from purely manual reviews toward automated, objective risk scoring.
