Smart Chair Posture Classification System

Developed an IoT-based smart chair posture classification system capable of monitoring sitting posture in real-time using machine learning algorithms. The system collected posture sensor data using ESP32 and pressure sensors, then processed and visualized the data through a Streamlit dashboard. Multiple ML models including Random Forest, KNN, and XGBoost were trained and evaluated for posture classification. The final XGBoost model achieved approximately 98% classification accuracy, including data preprocessing, feature engineering, visualization, model evaluation, and real-time monitoring system integration.
